# India in a Day

**One Date, Many Indias**

An interactive cultural calendar that showcases how different Indian states celebrate the same moment in time through their unique regional traditions.

## Overview

IN India in a Day addresses cultural ignorance by revealing the patterns and connections across India's diverse celebrations. Users can explore how festivals like Pongal, Lohri, and Makar Sankranti—celebrated under different names across states—actually mark the same astronomical event.

## Features

### Interactive 2026 Calendar
- Full-year calendar view with month-by-month navigation
- Festival dates highlighted for easy identification
- Clickable dates that navigate to dedicated festival pages

### Multi-State Festival Exploration
- Detailed information for festivals from multiple states on the same date
- Each festival includes:
  - Traditional rituals
  - Special foods
  - Cultural significance
  - Regional imagery

### Comparison System
- Select up to 4 festivals using checkboxes
- Side-by-side comparison view showing:
  - Common themes
  - Shared cultural elements
  - Ritual differences
  - Food traditions
  - Regional variations
- Cultural similarity scoring (0-100%)

### Educational Insights
- "Did You Know?" sections revealing connections between regional celebrations
- Explanations of astronomical and agricultural origins
- Pattern recognition across different cultural traditions

### Progress Tracking
- Visual progress badge showing festivals explored
- Counter tracking total festivals viewed
- Encouragement to discover more cultural connections

### User Interface
- Clean, modern design with gradient color schemes
- Responsive layout for desktop and mobile
- Smooth animations and transitions
- Intuitive navigation between calendar and festival pages

## Technical Implementation

### Frontend Architecture
Built with React 18 using component-based architecture for maintainability and reusability.

### Routing
React Router v6 enables seamless navigation between:
- Home page with calendar and date selector
- Individual date pages with full festival information

### Data Structure
Centralized festival data stored in JSON format containing:
- Festival metadata (name, state, date, theme)
- Cultural details (rituals, foods, significance)
- Connection information (common themes, shared elements, similarity scores)

### Styling
Vanilla CSS with:
- CSS Grid for responsive layouts
- Flexbox for component alignment
- Custom color variables inspired by Indian flag
- Google Fonts (Playfair Display for headings, Inter for body text)
